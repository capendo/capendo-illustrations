# capendo-illustrations

Per-passage scriptorium illustrations for [Capendo](https://github.com/capendo/capendo-ios).

Each `{nodeId}.png` is a Flux-generated ink-and-wash engraving in the Italian-Renaissance natural-history style, keyed by curriculum nodeId (e.g. `ch01-0.png` for "honey"). The curriculum is parallel across languages, so one image serves Italian, Spanish, German and French.

Served to the iOS app via **jsDelivr** — `https://cdn.jsdelivr.net/gh/capendo/capendo-illustrations@main/{nodeId-slug}.png`. Filenames replace `:` with `-` for filesystem + URL friendliness (so the API uses `ch01:0` while the file is `ch01-0.png`).

Generated by `capendo-worker/scripts/generate-illustrations.mjs` against the worker's token-gated `POST /v1/admin/illustration/generate` endpoint. The worker's `node_illustrations` D1 table records which nodeIds have art so `GET /v1/node/:id` can return the right `illustrationUrl`.
